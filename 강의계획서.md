## 2. 교과목 개요

본 과목은 딥러닝 기반 자연어처리(NLP)의 핵심 원리를 체계적으로 학습하고, 대규모 언어 모델(LLM)을 실무에 활용할 수 있는 역량을 기르는 것을 목적으로 한다. 신경망 기초부터 시작하여 Transformer 아키텍처, BERT/GPT 계열 모델, 파인튜닝, RAG, AI Agent, 모델 배포까지 현업에서 즉시 적용 가능한 기술을 단계적으로 다룬다.

---

## 3. 학습 목표

1. 딥러닝 신경망의 핵심 원리(역전파, 경사 하강법)를 이해하고 PyTorch로 구현할 수 있다.
2. Transformer 아키텍처의 Self-Attention, Multi-Head Attention, Positional Encoding을 구현하고 원리를 설명할 수 있다.
3. BERT, GPT 계열 사전학습 모델의 구조와 특성을 비교하고, Hugging Face 생태계를 능숙하게 활용할 수 있다.
4. LLM API 호출, 프롬프트 엔지니어링, Function Calling을 활용하여 AI 응용 시스템을 개발할 수 있다.
5. Full Fine-tuning, LoRA/QLoRA 등 파인튜닝 기법을 선택하고 적용할 수 있다.
6. RAG 파이프라인과 AI Agent를 설계하고 배포할 수 있다.

---

## 4. 강의 구성 철학

```
기초 압축 (4주) → LLM 핵심 (2주) → 중간고사 → 실무 기술 (5주) → 프로젝트 (2주) → 기말고사
```

- **기초는 빠르게, 실무는 깊게**: RNN은 개념 중심, Transformer부터 본격 구현
- **밑바닥 구현 → 프레임워크 활용**: 원리를 먼저 이해한 뒤 도구를 활용한다
- **취업 직결 기술 우선**: Agent, 배포, 평가 등 현업 필수 스킬 포함
- **직관 먼저, 수식 다음**: 모든 개념을 비유와 시각화로 먼저 이해하고, 이후 수학적 정의로 넘어간다

- Model Serving: FastAPI, vLLM, TGI(Text Generation Inference)
- 배포 환경: Docker 컨테이너화, HuggingFace Spaces
- 모니터링: Latency, Throughput, 오류율
- 비용 최적화 및 보안 고려사항

**B회차 실습**
- FastAPI로 모델 추론 API 서버 구현
- Docker 컨테이너화 및 배포
- 성능 벤치마크 및 평가 리포트 작성

### 14주차: 개인 프로젝트 워크숍
