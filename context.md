# 프로젝트 목표: 딥러닝 자연어처리 교재 집필

이 프로젝트는 **LLM 시대의 NLP 엔지니어링: 원리부터 배포까지**라는 부제의 학부생 대상 교재를 6개 전문 에이전트가 협력하여 집필·검증·변환하는 워크플로우 시스템이다.

- **대상 독자**: 컴퓨터공학/AI 전공 학부생 (3~4학년)
- **구성**: 15주차 (기초 4주 + LLM 2주 + 시험 + 실무 5주 + 프로젝트 2주 + 시험)
- **예상 분량**: 약 500페이지
- **최종 목차**: `contents.md` 참조
- **집필 철학**: 직관 먼저, 수식 다음 / 기초 빠르게, 실무 깊게 / 밑바닥 구현 → 프레임워크
